# Heroes

A turn-based tactical game in ClojureScript, DataScript and Rum.

<img src="extras/screenshot.png">

## Development

To get an interactive development environment run:

    ./repl.sh

To clean all compiled files:

    ./clean.sh

To create a production build run:

    ./min.sh
